## FFMPEG
**FFMPEG IS REQUIRED**

[Arch Linux]
`pacman -S ffmpeg`

[MacOS]
`brew install ffmpeg`
[Windows]
Install from https://www.gyan.dev/ffmpeg/builds/, place on path.

## Python Requirements
pyvidplayer2  `pip install pyvidplayer2`
